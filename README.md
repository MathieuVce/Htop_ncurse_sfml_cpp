# Htop_ncurse_sfml_cpp
* How to compile:
```sh
$ make re
```
* Launch with SFML:
```sh
$ ./MyGKrellm -s
```
* Launch with NCURSE:
```sh
$ ./MyGKrellm -n
```
* Get help:
```sh
$ ./MyGKrellm -h
```
* My SFML version:
    * 2.4.2
* My NCURSE version:
    * 6.1
